ROLE 02 / INNOVATOR

全地图都是黑雾时，
有人趟出一条路

当现有的工具、社会环境、沟通方式、学习方式，满足不了他预想中的那个更好的情形—— 他会自己动手把这个产品、这个流程、这套协作搭出来。

创新者与效仿者的核心区别，是对未来坚定的信心。

ChatGPT 出来之后，OpenAI 并没有做 coding agent；是 Boris（Anthropic）去做了，而且扎根了数年。很多人还没意识到这件事的时候，他已经开始披荆斩棘，为人类开辟新的道路。

— 这就是创新者

效仿者看到的是「原来做 coding agent 能赚很多钱」。他没有看到的是：全地图都是黑雾的时候， 有人趟出了那条路。效仿者也能赚到钱——但他没有创新者伟大。 这是创新者的引领作用，也是对人类的贡献。

但创新不必宏大

从 idea 到产品，
可能就是一个晚上

创新不一定发生在非常大的层面上，小一点的创新完全没问题。只要它是你自己的 idea。尤其在这个时代，在 AI 的加持下，从 idea 到产品，这个步骤可能只需要一个晚上。思考者、探索者，都是这一类角色的名字。
