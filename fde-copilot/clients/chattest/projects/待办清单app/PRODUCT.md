# 产品描述 · Product

> 客户：chattest ｜ 项目：待办清单app ｜ 交付物：待办清单（other）
> 一句话定位 / 目标用户 / 核心价值 / 关键场景

## 一句话定位
一个打开即用、无需注册的极简待办清单网页——加任务、打勾、删除，就这么简单。

## 目标用户
【调研假设·待确认】个人用户，想随手记录和勾销日常待办，追求零学习成本。（客户为「chattest」测试项目，用户画像待客户明确。）

## 核心价值
- **快**：打开网页即可写第一条任务，无登录无引导。
- **简单**：界面只有输入框 + 列表，三个动作一目了然。
- **不丢**：任务保存在本地，刷新不丢（假设，待确认）。

## 关键场景
1. **随手记**：用户想起一件事 → 打开页面 → 输入 → 回车 → 任务出现在列表顶部。
2. **勾销**：完成一件事 → 点击任务前的勾选框 → 任务显示为已完成（划线）。
3. **清理**：任务不再需要 → 点击删除按钮 → 任务从列表移除。
4. **回访**：关掉网页第二天再打开 → 之前的任务与完成状态还在（假设，待确认）。
