# 记账小程序 · 需求规格

> 客户：chattest ｜ 项目：ss2 ｜ 交付物：x（记账应用）

## 一句话定位
个人日常记账工具，快速记录每日支出、按分类统计开销。

## 目标用户
关注日常花费的个人用户；需要快速记账和分类查看。

## 核心功能
1. **添加支出记录** — 输入金额、选择分类、自动记录日期；提示"记录成功" ✓
2. **查看每日记录** — 列表展示当天已记的所有支出（金额、分类、时间）✓
3. **分类统计** — 按分类汇总支出，显示各分类总额和占比（如：饮食 200 元 / 40%）✓

## 范围
**范围内：**
- 单用户本地使用，不涉及注册登录
- 基础分类体系（待确认是否预定义 vs 用户自定义）
- 当日支出查看与分类统计

**范围外：**
- 多设备云端同步
- 数据导出/导入
- 历史查询、周/月对比统计（暂不做）
- 预算告警、定期推送等高级功能

## 技术方向
【假设·待确认】：
- **开发平台** — Web / 移动应用 / 桌面应用？（影响技术栈选型）

- **分类管理** — 系统预定义分类还是允许用户新增？
